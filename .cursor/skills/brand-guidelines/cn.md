# 人因品牌造型设计
## 概述

要获取Anthropic的官方品牌标识及风格资源，请使用此功能。

**关键词**：品牌推广、企业形象、视觉形象、后期处理、风格设计、品牌色彩、字体设计、阿尼普特品牌、视觉格式化、视觉设计
## 品牌指南
### 颜色
**主色调：**
- 深色：`#141413` - 主要文本及深色背景
- 浅色：`#faf9f5` - 浅色背景及深色文本
- 中灰色：`#b0aea5` - 次要元素
- 浅灰色：`#e8e6dc` - 轻微背景色
**主色调：**
- 橙色：`#d97757` - 主要强调色
- 蓝色：`#6a9bcc` - 次要强调色
- 绿色：`#788c5d` -  tertiary强调色
### 版式设计
- **标题字体**： 波普斯（若系统中未安装该字体，则会自动使用 Arial）
- **正文文本字体**： 莱拉（若系统中未安装该字体，则会自动使用 Georgia）
- **说明**： 为获得最佳效果，建议在您的环境中预先安装这些字体。
## 特点/功能
### 智能字体应用
- 将 Poppins 字体应用于标题（24 磅及以上）
- 将 Lora 字体应用于正文文本
- 若自定义字体不可用，则自动切换为 Arial 或 Georgia 字体
- 在所有系统中保持可读性
### 文本样式
- 标题（24 磅字体 +）：使用波普斯字体
- 正文：使用洛拉字体
- 根据背景智能选择颜色
- 保持文本的层级结构和格式不变
### 形状与点缀颜色
- 非文字形状采用强调色
- 依次使用橙色、蓝色和绿色作为强调色
- 既能保持视觉吸引力，又能符合品牌风格要求
## 技术细节
### 字体管理
- 当系统中安装有 Poppins 和 Lora 字体时，会使用这些字体
- 自动切换至 Arial 字体（标题）和 Georgia 字体（正文）
- 无需安装字体 - 可与现有系统字体配合使用
- 为获得最佳效果，请在您的环境中预先安装 Poppins 和 Lora 字体
### 颜色应用
- 采用 RGB 颜色值进行精确的品牌匹配
- 通过 python-pptx 的 RGBColor 类实现
- 在不同系统中保持颜色的一致性
