用高级向量数学重构画布拖拽缩放

现状分析

demo/d3drag.html 的 JS 逻辑约 170 行，核心问题：





8 方向 resize 用 if-chain：每个方向（t/b/l/r/tl/tr/bl/br）单独判断 dir.includes(...) 再分别计算 dx/dy 对 x/y/w/h 的影响



鼠标 vs 触摸重复：mousemove 和 touchmove 里各写了一遍完全相同的 resize 公式



状态是裸对象 {x, y, w, h}，没有向量语义

重构方案：向量数学统一公式

1. Vec2 类 — 二维向量

提供完整线性代数运算：

class Vec2 {
  constructor(x = 0, y = 0) { this.x = x; this.y = y; }
  add(v)  { return new Vec2(this.x + v.x, this.y + v.y); }
  sub(v)  { return new Vec2(this.x - v.x, this.y - v.y); }
  scale(s){ return new Vec2(this.x * s, this.y * s); }
  had(v)  { return new Vec2(this.x * v.x, this.y * v.y); } // Hadamard 积
  max(v)  { return new Vec2(Math.max(this.x,v.x), Math.max(this.y,v.y)); }
  round() { return new Vec2(Math.round(this.x), Math.round(this.y)); }
  static from(x, y) { return new Vec2(x, y); }
}

2. Rect 类 — 用 Vec2 表示矩形

class Rect {
  constructor(pos, size) { this.pos = pos; this.size = size; }
  // 通过方向向量一步 resize
  resized(delta, signVec, posVec, minSize) {
    const newSize = this.size.add(delta.had(signVec)).max(minSize);
    const newPos  = this.pos.add(this.size.sub(newSize).had(posVec));
    return new Rect(newPos.round(), newSize.round());
  }
  moved(delta) { return new Rect(this.pos.add(delta).round(), this.size); }
}

3. 方向签名表 — 用两个 Vec2 编码每个方向

核心数学思路：每个 handle 方向可以用两个向量完全描述：





signVec — delta 对 (w, h) 的影响系数（+1 / -1 / 0）



posVec — 当 size 收缩时 position 是否跟随（1 / 0）

方向   signVec       posVec        含义
─────────────────────────────────────────
br    (+1, +1)      (0, 0)        右下角拉大，原点不动
tl    (-1, -1)      (1, 1)        左上角拉，原点跟着缩
tr    (+1, -1)      (0, 1)        右上角，y原点跟随
bl    (-1, +1)      (1, 0)        左下角，x原点跟随
r     (+1,  0)      (0, 0)        右边
l     (-1,  0)      (1, 0)        左边
b     ( 0, +1)      (0, 0)        下边
t     ( 0, -1)      (0, 1)        上边

这样，resize 计算变成 一行向量公式，完全消除 if-chain：

const DIRS = {
  tl: { sign: Vec2.from(-1,-1), pos: Vec2.from(1,1) },
  tr: { sign: Vec2.from(+1,-1), pos: Vec2.from(0,1) },
  bl: { sign: Vec2.from(-1,+1), pos: Vec2.from(1,0) },
  br: { sign: Vec2.from(+1,+1), pos: Vec2.from(0,0) },
  t:  { sign: Vec2.from( 0,-1), pos: Vec2.from(0,1) },
  b:  { sign: Vec2.from( 0,+1), pos: Vec2.from(0,0) },
  l:  { sign: Vec2.from(-1, 0), pos: Vec2.from(1,0) },
  r:  { sign: Vec2.from(+1, 0), pos: Vec2.from(0,0) },
};

4. 统一指针抽象 — 消除 mouse/touch 重复

function ptrPos(e) {
  const p = e.touches ? e.touches[0] : e;
  return Vec2.from(p.clientX, p.clientY);
}

用 pointerdown / pointermove / pointerup 事件（现代浏览器原生支持）统一鼠标和触摸，或至少用 ptrPos() 抽取坐标后走同一条 handler 逻辑。

5. 事件处理：只需一个 move handler

function onPointerMove(e) {
  const cur = ptrPos(e);
  if (interaction.type === 'move') {
    state = interaction.snap.moved(cur.sub(interaction.startPt));
  } else {
    const { sign, pos } = DIRS[interaction.dir];
    state = interaction.snap.resized(cur.sub(interaction.startPt), sign, pos, MIN_SIZE);
  }
  applyState(state);
}
